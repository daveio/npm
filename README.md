# dave.io

Meta package for Dave Williams

With thanks to [zachleat](https://github.com/zachleat)

## Installation

* [`dave.io` on npm](https://www.npmjs.com/package/dave.io)

```sh
npm install dave.io
```

## Usage

### CLI

```sh
npx dave.io
```

### Programmatic

```js
import dave from "dave.io";

// or in CommonJS
// const dave = await import("dave.io");

dave();
```
